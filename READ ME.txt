-CRUD Console-
It is a small console application designed to manage users. 
It is simple and was created to explain the fundamentals 
of databases and the use of CRUD operations (Create, Read, Update, Delete).

##Prerequisites
- .NET SDK 6.0 or higher
- SQL Server, SQL Management Studio, or any database management tool
- Git
- Visual Studio 2019 or higher

##Installation
1. Clone this repository: 
  git clone https://github.com/FrannyRC96/CRUD-Console/tree/master/source/repos/CRUD%20Console
2. Open the project in Visual Studio
3. Open an SQL Server window
4. In the repository, there is a file named: 
"TableUsuariosQuery" Execute it in SQL Server. If necessary, create a new database.
5. Configure the connection string in the Program.cs file:
	static string servername = "YourServer";  
	static string databasename = "YourDatabase";  

##Usage
The application is straightforward and intuitive, with options numbered 1 to 5. It uses CRUD functionalities.
